# **App Name**: Mohamed Aref Portfolio

## Core Features:

- Portfolio Display: Display portfolio items in a grid layout with filtering options.
- Multi-Language Support: Implement a multi-language support system for Arabic, English, and French.
- Contact Form: Create a contact form for inquiries. also floating button with social media icons.
- PWA Support: Progressive Web App (PWA) functionality for offline access and app-like experience.
- AI-Powered Feedback Tool: AI-powered 'Vibe Check' tool that analyzes uploaded code snippets and video edits, providing constructive feedback suggestions based on industry standards. The tool suggests improvements that enhance the quality of code or edits and save time for users.
- SQLite data setup: Implement a database schema to support different kinds of data used by this application

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) for a professional and trustworthy feel.
- Background color: Very light indigo (#F0F2F9), almost white, to provide a clean and modern backdrop.
- Accent color: Cyan (#00BCD4) to highlight interactive elements and calls to action.
- Body font: 'Inter', a sans-serif font known for its clean and modern appearance, providing excellent readability. Headline Font: 'Space Grotesk', a geometric, attention-grabbing sans-serif for section titles.
- Responsive design that adapts to different screen sizes using a grid-based layout.
- Use Material Design icons for a consistent and modern look.
- Subtle animations and transitions for a smooth user experience.
